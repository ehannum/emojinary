Rules:

MOJIES is a same-room party game. That means it is meant to be played together with lots of friends in the same place. You will each need your phone to play.

The main player picks from a list of 3 randomly selected movies and has to describe or act it out using ONLY EMOJI. The other players try to guess the movie at the same time. Once a player guesses correctly, the main player presses the "correct" button and awards that player their points. Points for a round are based on the number of hints given before a winner is chosen. Turns progress from there in the order players joined.

Important note: Technically, MOJIES will work remotely and accept typed guesses in the input box at the bottom but this is not recommended. You'll probably have to make heavy use of the hint system to make sure people are typing the title exactly correctly. A solution to this is in the works.
